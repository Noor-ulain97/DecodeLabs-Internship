# DecodeLabs-Internship

**📝 To-Do List App (Python)**

A simple and efficient command-line To-Do List application built using Python.
This project helps users manage daily tasks with features like adding, viewing, updating, and deleting tasks.


----------------------------
**Features**
➕ Add new tasks
📋 View all tasks
✅ Mark tasks as completed
🗑️ Delete tasks
💾 Data saved using JSON (persistent storage)
📌 Predefined tasks included


-------------------------
**🛠️ Technologies Used**
Python
JSON (File Handling)
